Core Utility API

문서의 종류, 업무 규칙에 상관없이 독립적으로 작동하는 레이어

  • POST /api/v1/utils/extract
  • POST /api/v1/utils/classify

Workflow API

특정 문서로부터 추출된 데이터를 바탕으로 실제 업무 요건을 판정하고 관리하는 레이어

  • PUT /api/v1/workflow/schema
  • POST /api/v1/workflow/label
  • POST /api/v1/job/batch
  • GET /api/v1/jobs/{id}/status
  • GET /api/v1/jobs/{id}/download
