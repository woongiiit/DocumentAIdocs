7 결과 리포트 다운로드 (Download)

  • Endpoint: GET /api/v1/jobs/{id}/download
  • Description: 최종 판정 리포트와 분류된 파일을 다운로드합니다.
  • Request: Path Parameter (job_id)
  • Response: application/json

  "report_url": "https://api.domain.com/static/report_1234.xlsx",
