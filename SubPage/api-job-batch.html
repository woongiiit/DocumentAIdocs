POST /api/v1/job/batch

  • Method: POST
  • Endpoint: /api/v1/jobs/batch
  • Description: 등록된 룰에 따라 대량 문서 일괄 판정을 시작합니다.
  • Request: multipart/form-data
    • files : (List of Binaries) 판정대상 파일들
    • workflow_id : (String) 적용할 워크플로우 ID
  • Response: {"job_id":"job_uuid_1234","status":"pending"}
