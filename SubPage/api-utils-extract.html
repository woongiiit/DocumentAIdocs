1) 문서 스키마 추출 (Extract)

  • Method: POST
  • Endpoint: /api/v1/utils/extract
  • Description: 단일 PDF/이미지 파일에서 텍스트 데이터와 시각적 위치(Bounding Box)를 추출합니다.
  • Request: multipart/form-data
    • file : (Binary) PDF 또는 이미지
  • Sample Response:
JSON [ ] 
{
  "file_name": "pdf_file_name.pdf",
  "elements": [
    { "attribute": "Name", "value": "이용희", "location": [100, 250, 80, 20] },
    { "attribute": "Total Score", "value": "850", "location": [100, 200, 50, 20] }    
  ]
}
