6 작업 상태 조회 (Status Check)

  • Endpoint: GET /api/v1/jobs/{id}/status
  • Description: 비동기 작업의 현재 진행률과 상태를 확인합니다.
  • Request: Path Parameter (job_id)
  • Response: application/json
JSON [ ] 
{
  "progress_percent": 75,
  "status": "processing",
  "total_files": 200,
  "completed_files": 150
}
