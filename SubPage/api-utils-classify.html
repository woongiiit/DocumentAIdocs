2) 문서 자동 분류 (Classify)

  • Endpoint: POST /api/v1/utils/classify
  • Description: 여러 파일을 업로드하여 유사도 기반으로 그룹화(Clustering) 합니다.
  • Request: multipart/form-data
    • files : (List of Binaries) 다중 문서 파일
  • Response: application/json
JSON [ ] 
{
  "job_id": "batch_unique_01",
  "results": [
    { "file_name": "doc_a.pdf", "cluster_id": "Category1", "confidence": 0.98 },
    { "file_name": "doc_b.pdf", "cluster_id": "Category2", "confidence": 0.95 }
  ]
}
