3 스키마 수정 및 확장 (Schema Update)

  • Endpoint: PUT /api/v1/workflow/schema/{schema_id}
  • Description: AI가추출한 필드명이나 데이터 타입을 사용자 교정합니다.
  • Request: application/json
JSON [ ] 
{
  "modified_elements": [
    {
      "original_attribute": "Total Score",
      "new_attribute": "성적",
      "type": "..."
    }
  ]
}
  • Response: {"status":"success","message":"Schema updated"}
