5 배치 작업 실행 (Batch Execute)

  • Endpoint: POST /api/v1/jobs/batch
  • Description: 등록된 롤에 따라 대량 문서 일괄 판정을 시작합니다.
  • Request: multipart/form-data
    • files : (List of Binaries) 판정 대상 파일들
    • workflow_id : (String) 적용할 워크플로우 ID
  • Response: {"job_id":"job_uuid_1234","status":"pending"}
